Help & Keyboard Shortcuts

Keyboard shortcuts for viewer and page actions.

Enter / Click Open selected image
+ Zoom in
- Zoom out
0 / ⟲ Reset zoom
← / → Previous / Next image
Esc Close viewer or exit fullscreen
F / ⤢ Toggle image fullscreen
T Toggle theme (light / dark)
H Open Help modal
Drag Pan when image is zoomed
Wheel Zoom (when viewer open)
